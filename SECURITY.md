# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Whendows, please report it responsibly using [GitHub's private vulnerability reporting](https://github.com/GSonofNun/Whendows/security/advisories/new).

**Do not** open a public issue for security vulnerabilities.

## What to Report

- Vulnerabilities in the Whendows application itself
- Security issues in how Whendows interacts with Claude CLI or Codex CLI
- Data exposure or privilege escalation concerns

## What Not to Report

- Vulnerabilities in Claude CLI, Codex CLI, or their underlying services — report those directly to [Anthropic](https://www.anthropic.com/responsible-disclosure) or [OpenAI](https://openai.com/security)
- General bugs or feature requests — use [Issues](https://github.com/GSonofNun/Whendows/issues) instead
